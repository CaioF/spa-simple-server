# spa-simple-server
Express server created for qoollo.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 

### Prerequisites

```
Node.js version >= 10
npm version >= 6
```

### Installing

```
git clone 
cd spa-simple-server/
npm install
npm run dev
```
